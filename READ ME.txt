Two ways to run the maxwell nutrition mobile app


1) If you use github zip to run the project follow these steps

	* Download the pods folder : https://drive.google.com/file/d/1I57vJ_q0jUPjcYE2Z1hv7RxXbMXzFDjq/view?usp=share_link
	* Place the pods folder inside the "MaxwellNutrition-main folder"
	* Run the MaxwellNutritionApp.xcworkspace" file

2) if you use google drive to get the project follow these steps below.

	* Download the complete project file : https://drive.google.com/file/d/1iHd0YQCoil9YjOM0nk-DAT89wlAnyTk7/view?usp=share_link
	* Run the MaxwellNutritionApp.xcworkspace" file
